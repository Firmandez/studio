# **App Name**: Sahid Tani

## Core Features:

- Interactive About Us: A dynamic 'About Us' section detailing Sahid Tani's mission and values using engaging visuals and animations to convey the company's story.
- Product Catalog: A visually appealing 'Products' catalog displaying high-quality images and descriptions of plant seedlings available from Sahid Tani. don't Use any AI tool to determine similar or matching seedlings using available image data.
- Image Gallery: An engaging 'Gallery' showcasing the cultivation process and the variety of seedlings, enhancing user interest.
- Inquiry Form: A 'Contact Us' page with a user-friendly inquiry form that allows visitors to submit questions and requests directly to Sahid Tani.

## Style Guidelines:

- Primary color: Fresh green (#8FBC8F), inspired by seedling leaves.
- Background color: Light, desaturated earthy brown (#F5F5DC) to evoke nature.
- Accent color: Soft yellow (#FFEC8B) to represent sunlight.
- Body and headline font: 'PT Sans', a humanist sans-serif providing a modern, clean look while retaining warmth and personality.
- Use natural and simple icons related to plants, leaves, and growth, complementing the earthy theme.
- A clean, modern layout with ample whitespace to emphasize content and imagery, promoting a contemporary user experience.
- Subtle animations like fade-ins and transitions to add visual interest without being distracting, creating an interactive and engaging experience.
